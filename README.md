- NodeJS
- MongoDB
- ExpressJS
- EJS
- Bootstrap


## Functionality 
- User/Employee signUp and signIn
- Add student 
- Delete student
- Allocate added student to the interview
- Deallocate the student from an interview
- Update user profile
- Download the CSV report
- Find latest jobs on jobs page
